### Set Incentives Pool

```solidity
function setIncentivesPool(
    address incentivesPool_,
    IncentivesConfig calldata incentivesConfig_
) external onlyOwner
```

**Requirements:**
- Can only be called BEFORE `boardOpenAt`
- Must provide `IncentivesConfig` with incentive curve parameters
